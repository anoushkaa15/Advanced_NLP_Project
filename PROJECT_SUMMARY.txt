═══════════════════════════════════════════════════════════════════════════════════
🎯 HOW THE CHATBOT WORKS
═══════════════════════════════════════════════════════════════════════════════════

Pipeline:
  User Input (Spanish): "Tengo fiebre y tos"
         ↓
  [1] Language Detection (langdetect)
      → Detects: Spanish (es)
         ↓
  [2] Translation (googletrans)
      → Translates to English: "I have fever and cough"
         ↓
  [3] TF-IDF Vectorization (sklearn)
      → Converts to numerical representation
         ↓
  [4] Similarity Matching (cosine_similarity)
      → Compares with all 169,888 dataset entries
      → Finds top matches:
        • pneumonia (87% match)
        • acute bronchitis (84% match)
        • infection (76% match)
         ↓
  [5] Department Lookup
      → pneumonia → Pulmonology
      → acute bronchitis → Pulmonology
         ↓
  [6] Response Generation
      → "Based on your symptoms, pneumonia is most likely."
         ↓
  [7] Back-translation (googletrans)
      → Translates to Spanish
         ↓
  Output (Spanish): "Basado en sus síntomas, la neumonía es más probable."

Speed: ~2 seconds total
Confidence: 87.5%

❓ FREQUENTLY ASKED QUESTIONS
═══════════════════════════════════════════════════════════════════════════════════

Q: Which Python version do I need?
A: Python 3.7 or higher recommended

Q: Can I use this for real medical diagnosis?
A: NO! This is for demonstration/educational purposes only.
   Always consult actual medical professionals for diagnosis.

Q: Why is the first run slow?
A: Building the TF-IDF matrix takes time. Subsequent runs are fast.

Q: Can I improve the accuracy?
A: Yes! Use BERT embeddings instead of TF-IDF (see documentation).

Q: How many languages does it support?
A: 100+ languages (anything Google Translate supports).

Q: Can I deploy this as a web app?
A: Yes! See deployment sections in documentation.

Q: What if a symptom is not in the database?
A: The chatbot will still try to find similar symptoms.
   If no match is found, it returns a generic response.

Q: Can I modify the disease-department mappings?
A: Yes! Edit the DISEASE_DEPARTMENT_MAPPING in add_department_column.py
