EXAMPLE 2: Python API (Single Query)
─────────────────────────────────────
from medical_chatbot_api import MedicalChatbotAPI

api = MedicalChatbotAPI()
result = api.analyze_symptoms("I have joint pain and swelling")

print(f"Disease: {result['predicted_disease']}")
print(f"Department: {result['predicted_department']}")
print(f"Confidence: {result['confidence']:.1%}")

Output:
Disease: arthritis
Department: Orthopedics
Confidence: 82.3%

EXAMPLE 3: Python API (Multiple Queries)
────────────────────────────────────────
symptoms_list = [
    "fever and cough",
    "chest pain",
    "joint pain"
]

results = api.analyze_symptoms_batch(symptoms_list)
for result in results:
    print(f"{result['predicted_disease']} ({result['predicted_department']})")

📋 API FUNCTIONS
────────────────────────────────────────────────────────────────────────────────

1. analyze_symptoms(symptoms: str) → Dict
   Single symptom analysis
   Returns: disease, department, confidence, alternatives

2. analyze_symptoms_batch(symptoms_list: List[str]) → List[Dict]
   Batch processing multiple queries

3. get_statistics() → Dict
   Dataset info: total records, diseases, departments

4. search_diseases(disease_name: str) → List[Dict]
   Find diseases by name

5. get_department_diseases(department: str) → List[str]
   Get all diseases in a specific department


🔧 CONFIGURATION
────────────────────────────────────────────────────────────────────────────────

Edit medical_chatbot.py to adjust:

SIMILARITY_THRESHOLD = 0.3
  → Minimum match score (0-1)
  → Increase for stricter matching
  → Decrease for more liberal matching

TOP_N_MATCHES = 3
  → Number of alternative results to show
  → Show more matches for user choice
  → Show fewer for cleaner output

❓ TROUBLESHOOTING
────────────────────────────────────────────────────────────────────────────────

"ModuleNotFoundError: langdetect"
  → pip install langdetect

"ModuleNotFoundError: googletrans"
  → pip install googletrans

"ModuleNotFoundError: sklearn"
  → pip install scikit-learn

"Dataset not found"
  → Make sure symptom_sentence_dataset_with_department.csv exists
  → Run add_department_column.py if not present

"No matches found"
  → Try more descriptive symptoms
  → Lower SIMILARITY_THRESHOLD in code

"Slow response"
  → First run builds TF-IDF vectors (30-60 seconds)
  → Subsequent queries are <2 seconds
